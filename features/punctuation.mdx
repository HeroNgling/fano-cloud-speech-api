## Overview

Automatic punctuation makes your transcripts immediately readable and ready for downstream applications. When enabled, the API intelligently adds punctuation marks to the transcribed text.

## Enable Feature

By default, `enableAutomaticPunctuation` is set to `false`. You can enable the feature based on your preference.

**Example Configuration:**

```json
{
  "config": {
    "languageCode": "yue",
    "encoding": "LINEAR16",
    "enableAutomaticPunctuation": true
  }
}
```

## Results

Once applied, punctuation will appear in the transcript automatically.

| Source | Before Punctuation | After Punctuation |
|--------|-------------------|-------------------|
| 誒你可唔可以講廣東話㗎 | 誒你可唔可以講廣東話㗎 | 誒，你可唔可以講廣東話㗎？ |

<Note>
Punctuation is applied intelligently based on the context and language of the speech, supporting Cantonese, Mandarin, and English.
</Note>
