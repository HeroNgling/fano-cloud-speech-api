# Punctuation

The Fano Speech API automatically adds punctuation marks to your transcripts, making them more readable and easier to process. This feature uses advanced NLP models to intelligently place punctuation based on speech patterns and context.

## Supported Punctuation

| Mark | Description | Example |
|------|-------------|---------|
| `.` | Period | End of declarative sentences |
| `,` | Comma | Pauses, lists, clauses |
| `?` | Question mark | Interrogative sentences |
| `!` | Exclamation mark | Emphatic statements |
| `:` | Colon | Introduction to lists/quotes |
| `;` | Semicolon | Related independent clauses |

## Enabling Punctuation

Punctuation is enabled by default. You can explicitly control it:

### Pre-recorded Audio

<CodeGroup>

```python Python
response = requests.post(
    "https://api.fano.ai/v1/transcribe",
    headers={"Authorization": "Bearer your_api_key"},
    json={
        "audio_url": "https://example.com/audio.wav",
        "punctuation": True,  # Enable punctuation (default)
        "capitalize": True    # Enable auto-capitalization (default)
    }
)
```

```javascript JavaScript
const response = await fetch('https://api.fano.ai/v1/transcribe', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    audio_url: 'https://example.com/audio.wav',
    punctuation: true,
    capitalize: true
  })
});
```

```bash cURL
curl -X POST "https://api.fano.ai/v1/transcribe" \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "audio_url": "https://example.com/audio.wav",
    "punctuation": true,
    "capitalize": true
  }'
```

</CodeGroup>

### Streaming Audio

Include in your configuration message:

```json
{
  "type": "config",
  "language": "auto",
  "punctuation": true,
  "capitalize": true
}
```

## Parameters

<ParamField body="punctuation" type="boolean" default="true">
  Enable or disable automatic punctuation.
</ParamField>

<ParamField body="capitalize" type="boolean" default="true">
  Enable or disable automatic capitalization (sentence start, proper nouns).
</ParamField>

## Output Examples

### With Punctuation Enabled

```json
{
  "transcript": "Hello, this is a test. How are you doing today? I hope everything is going well!"
}
```

### With Punctuation Disabled

```json
{
  "transcript": "hello this is a test how are you doing today i hope everything is going well"
}
```

## Mixed-Language Support

When transcribing mixed-language audio (e.g., English with Cantonese), the API intelligently applies the appropriate punctuation style:

```json
{
  "transcript": "今日我哋講吓 API integration, okay? 呢個 feature 好 useful。"
}
```

<Info>
The punctuation model automatically detects language switches and applies contextually appropriate punctuation.
</Info>

## Best Practices

<Tip>
**When to Disable Punctuation:**
- When feeding output to another NLP system that handles punctuation
- For voice command applications where punctuation is irrelevant
- When you need raw word-by-word output
</Tip>

<Warning>
**Punctuation Accuracy Note:**
While our punctuation model is highly accurate, it may occasionally make errors, especially with:
- Unusual speech patterns
- Rapid language switching
- Technical or domain-specific content

For critical applications, we recommend reviewing and editing the punctuated output.
</Warning>
