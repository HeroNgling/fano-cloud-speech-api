# Overview

The Fano Speech API provides accurate and efficient speech recognition capabilities with advanced features including multilingual support, speaker diarization, and real-time transcription. Built specifically for enterprise applications, our API excels in processing Cantonese, Mandarin, and English with industry-leading accuracy.

## Key Features

<CardGroup cols={2}>
  <Card title="High-accuracy Speech Recognition" icon="microphone">
    Industry-leading accuracy for multiple languages with over 90% accuracy rate
  </Card>
  <Card title="Multilingual Support" icon="language">
    Mixed-language detection for Cantonese, Mandarin, and English
  </Card>
  <Card title="Speaker Diarization" icon="users">
    Automatically identify and separate up to 8 different speakers
  </Card>
  <Card title="Custom Keyword Biasing" icon="bullseye">
