# Release Notes

Stay up to date with the latest features, improvements, and bug fixes for the Fano Speech API.

## How We Version

We follow semantic versioning for our API:

- **Major versions** (v1, v2): Breaking changes that require code updates
- **Minor versions** (v1.1, v1.2): New features, backward compatible
- **Patches**: Bug fixes and improvements

<Info>
The current API version is **v1**. All endpoints use the `/v1/` prefix.
</Info>
