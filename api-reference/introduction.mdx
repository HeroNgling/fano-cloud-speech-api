## Base URL

All API requests should be made to:

```
https://studio.fano.ai
```

## Authentication

All requests to the Fano Speech API must be authenticated using your license key. Include the key in the request header:

```bash
Fano-License-Key: YOUR_API_KEY
```

<Warning>
Keep your API key secure. Do not share it publicly or commit it to version control.
</Warning>

## Response Format

All API responses are returned in JSON format. Successful responses will contain the requested data, while error responses will include an error code and message.

## Interactive API Playground

Each API endpoint below includes an interactive playground where you can:

- View request/response schemas
- See example requests in multiple languages (cURL, Python, JavaScript)
- Test API calls directly (requires your API key)

<Note>
To use the API playground, enter your `Fano-License-Key` in the authentication section.
</Note>
