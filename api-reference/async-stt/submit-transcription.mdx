<Note>
This is the first step of the transcription workflow. After submitting, use the returned `operation_id` to poll for results.
</Note>

## Workflow

```mermaid
sequenceDiagram
    participant Client
    participant API as Fano API
    
    Client->>API: POST /api/v1/transcript
    API-->>Client: Returns operation_id
    
    loop Poll for results
        Client->>API: GET /api/v1/transcript/{operation_id}
        API-->>Client: Returns status (done: false/true)
    end
    
    Note over Client,API: When done: true, response contains transcript
```

## Audio Input Options

You can provide audio in two ways:

| Method | Field | Description |
|--------|-------|-------------|
| **URI** | `audio.uri` | Public URL to your audio file |
| **Base64** | `audio.content` | Base64-encoded audio content |

<Tip>
For large files, we recommend using a URI to avoid request size limits.
</Tip>
