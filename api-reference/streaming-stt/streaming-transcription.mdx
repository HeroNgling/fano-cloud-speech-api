## Overview

The Streaming Transcription API enables real-time speech-to-text transcription over a WebSocket connection. 

### Use Cases

- **Live Agent Assist** - Transcribe contact center calls in real-time
- **Meeting Transcription** - Live notetaking for Cantonese, Mandarin, and English
- **Voice Assistants** - Real-time voice command processing
- **Captioning** - Live subtitles for broadcasts

## Connection Flow

1. **Connect** - Establish WebSocket connection with `Fano-license-key` header
2. **Configure** - Send configuration message with language, sample rate, encoding
3. **Stream Audio** - Send Base64-encoded audio chunks
4. **Receive Results** - Get real-time transcription results
5. **Close** - Send EOF message to end session

## Code Examples

<Tabs>
  <Tab title="Python">
```python
import asyncio
import websockets
import json
import base64

API_KEY = 'YOUR_FANO_API_KEY'
URL = "wss://app.fano.ai/api/v1/speech-to-text/streaming-transcript"

async def stream_audio():
    headers = {"Fano-license-key": API_KEY}
    
    async with websockets.connect(URL, extra_headers=headers) as ws:
        # Send configuration
        await ws.send(json.dumps({
            "config": {
                "languageCode": "yue-x-auto",
                "sampleRateHertz": 16000,
                "encoding": "LINEAR16",
                "enableAutomaticPunctuation": True,
                "interimResults": True
            }
        }))
        
        # Send audio chunks
        with open("audio.raw", "rb") as f:
            while chunk := f.read(3200):
                await ws.send(json.dumps({
                    "audioContent": base64.b64encode(chunk).decode()
                }))
                
                # Receive results
                response = await ws.recv()
                print(json.loads(response))
        
        # Send EOF
        await ws.send(json.dumps({"eof": True}))

asyncio.run(stream_audio())
```

  <Tab title="JavaScript">
```javascript
const WebSocket = require('ws');

const ws = new WebSocket('wss://app.fano.ai/api/v1/speech-to-text/streaming-transcript', {
  headers: { 'Fano-license-key': 'YOUR_API_KEY' }
});

ws.on('open', () => {
  // Send configuration
  ws.send(JSON.stringify({
    config: {
      languageCode: 'yue-x-auto',
      sampleRateHertz: 16000,
      encoding: 'LINEAR16'
    }
  }));
});

ws.on('message', (data) => {
  const result = JSON.parse(data);
  if (result.results) {
    console.log(result.results[0].alternatives[0].transcript);
  }
});
```

## Supported Languages

| Code | Language |
|------|----------|
| `yue` | Cantonese |
| `cmn` | Mandarin Chinese |
| `en` | English |
| `yue-x-auto` | Multilingual (auto-detect) |
