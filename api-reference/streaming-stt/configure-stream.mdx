---
title: "Configure Stream"
openapi: "POST /streaming-transcript/configure"
description: "Initialize the transcription session with your desired settings"
---

<Note>
Send this message **immediately** after the WebSocket connection is established.
</Note>

## Configuration Options

### Required Fields

| Field | Type | Description |
|-------|------|-------------|
| `config.languageCode` | string | Language: `yue`, `cmn`, `en`, `yue-x-auto` |
| `config.sampleRateHertz` | integer | Sample rate: 8000-48000 (optimal: 16000) |
| `config.encoding` | string | Audio encoding: `LINEAR16` or `MULAW` |

### Optional Fields

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `config.enableAutomaticPunctuation` | boolean | `false` | Add punctuation to results |
| `config.interimResults` | boolean | `false` | Return interim (tentative) results |
| `config.speechContexts.keywords` | string[] | `[]` | Keywords for biasing (max 1000) |
| `config.audioChannelCount` | integer | `1` | Number of audio channels |
| `config.enableSeparateRecognitionPerChannel` | boolean | `false` | Separate recognition per channel |

## Examples

<CodeGroup>

```json Basic Configuration
{
  "config": {
    "languageCode": "yue",
    "sampleRateHertz": 16000,
    "encoding": "LINEAR16"
  }
}
```

```json With Punctuation & Interim Results
{
  "config": {
    "languageCode": "yue-x-auto",
    "sampleRateHertz": 16000,
    "encoding": "LINEAR16",
    "enableAutomaticPunctuation": true,
    "interimResults": true
  }
}
```

```json With Keyword Biasing
{
  "config": {
    "languageCode": "en",
    "sampleRateHertz": 16000,
    "encoding": "LINEAR16",
    "enableAutomaticPunctuation": true,
    "interimResults": true,
    "speechContexts": {
      "keywords": ["Fano", "transcription", "speech recognition"]
    }
  }
}
```

</CodeGroup>

## Best Practices

<AccordionGroup>
  <Accordion title="Choosing a Language Code">
    - Use specific codes (`yue`, `cmn`, `en`) when you know the language in advance for better accuracy
    - Use `yue-x-auto` for multilingual conversations or when language may switch mid-stream
  </Accordion>
  <Accordion title="Sample Rate Selection">
    - **16000 Hz** is optimal for speech recognition
    - If your audio source uses a different native rate, use that instead of resampling
    - Resampling can introduce artifacts that reduce accuracy
  </Accordion>
  <Accordion title="Interim Results">
    - Enable `interimResults` for real-time feedback in UI applications
    - Interim results have `isFinal: false` and may change
    - Wait for `isFinal: true` before considering the transcription complete
  </Accordion>
</AccordionGroup>
