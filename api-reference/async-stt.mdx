## Submit Audio for Transcription

<ParamField header="Fano-License-Key" type="string" required>
  Your unique API key for authentication.
</ParamField>

### Endpoint

```
POST https://studio.fano.ai/api/v1/transcript
```

This is the first step of the transcription process. Send your audio file (as base64 `content` or a public `uri`) and configuration. The API will validate the request and return an `Operation` object with a unique `name` (the operation ID).

### Request Body

<ParamField body="config" type="object" required>
  Configuration for the transcription job.

  <Expandable title="config properties">
    <ParamField body="languageCode" type="string" required>
      The language code for transcription. Options: `yue`, `cmn`, `en`, `yue-x-auto`
    </ParamField>
    <ParamField body="encoding" type="string" required>
      Audio encoding format. Options: `LINEAR16`, `MULAW`, `MP3`, `FLAC`, `OGG_OPUS`, `AAC`
    </ParamField>
    <ParamField body="enableAutomaticPunctuation" type="boolean">
      Enable automatic punctuation in the transcript. Default: `false`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="audio" type="object" required>
  Audio data to transcribe. You must provide exactly one of `content` or `uri`.

  <Expandable title="audio properties">
    <ParamField body="uri" type="string">
      Public URL of the audio file to transcribe.
    </ParamField>
    <ParamField body="content" type="string">
      Base64-encoded audio content.
    </ParamField>
  </Expandable>
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST https://studio.fano.ai/api/v1/transcript \
  -H "Fano-License-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "config": {
      "languageCode": "yue-x-auto",
      "encoding": "LINEAR16",
      "enableAutomaticPunctuation": true
    },
    "audio": {
      "uri": "https://example.com/audio.wav"
    }
  }'
```

```python Python
import requests

response = requests.post(
    "https://studio.fano.ai/api/v1/transcript",
    headers={
        "Fano-License-Key": "YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "config": {
            "languageCode": "yue-x-auto",
            "encoding": "LINEAR16",
            "enableAutomaticPunctuation": True
        },
        "audio": {
            "uri": "https://example.com/audio.wav"
        }
    }
)
print(response.json())
```

```javascript JavaScript
const response = await fetch("https://studio.fano.ai/api/v1/transcript", {
  method: "POST",
  headers: {
    "Fano-License-Key": "YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    config: {
      languageCode: "yue-x-auto",
      encoding: "LINEAR16",
      enableAutomaticPunctuation: true
    },
    audio: {
      uri: "https://example.com/audio.wav"
    }
  })
});
const data = await response.json();
console.log(data);
```

</CodeGroup>

### Response

<ResponseField name="name" type="string">
  The unique operation ID for this transcription job.
</ResponseField>

<ResponseField name="done" type="boolean">
  Whether the transcription job is complete.
</ResponseField>

<ResponseField name="metadata" type="object">
  <Expandable title="metadata properties">
    <ResponseField name="requestTime" type="string">

    </ResponseField>
    <ResponseField name="startTime" type="string">

## Get Transcription Status or Results

### Endpoint

```
GET https://studio.fano.ai/api/v1/transcript/{operation_id}
```

Poll this endpoint to check the status of a transcription job.

- If `done: false`, the job is still in progress.
- If `done: true` and an `error` object is present, the job failed.
- If `done: true` and a `response` object is present, the job succeeded and results are available.

### Path Parameters

<ParamField path="operation_id" type="string" required>
  The unique ID of the operation to check. This is the `name` field returned from the POST request.
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X GET https://studio.fano.ai/api/v1/transcript/2a0ffea4-f98b-47dc-a55b-963a2e01ba29 \
  -H "Fano-License-Key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://studio.fano.ai/api/v1/transcript/2a0ffea4-f98b-47dc-a55b-963a2e01ba29",
    headers={"Fano-License-Key": "YOUR_API_KEY"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://studio.fano.ai/api/v1/transcript/2a0ffea4-f98b-47dc-a55b-963a2e01ba29",
  {
    headers: { "Fano-License-Key": "YOUR_API_KEY" }
  }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

### Error Responses

| Status Code | Description |
|-------------|-------------|
| `401` | Unauthorized - Check your `Fano-License-Key` |
| `404` | Not Found - The specified operation_id does not exist or has expired |
